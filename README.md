# Kugou Music

资源版权归kugou所有。

## Run

install dependences

```shell
sudo apt-get install  libqt5multimedia5-plugins
```

if no `libqt5multimedia5-plugins` installed, there is a error.

```shell
defaultServiceProvider::requestService(): no service found for - "org.qt-project.qt.mediaplayer"
```
